# Cards Against Rust
A [Cards Against Humanity](http://cardsagainsthumanity.com/) clone written in Rust. 

Loosely based off of [Cards Against Megukas](https://github.com/lalcmellkmal/cam).

## Quickstart
 * `docker-compose build`
 * `docker-compose up -d`
